# WildBloom

A full-stack e-commerce application for a flower shop built with Java, Spring Boot, PostgreSQL, and vanilla JavaScript.

## Overview
WildBloom is a layered web application with a Spring Boot backend and a JavaScript frontend. It supports product catalog browsing, shopping cart management, order placement, and role-based administration workflows.

## Key Features
- Role-based access control with CUSTOMER, EMPLOYEE, and ADMINISTRATOR roles
- Product and category management
- Shopping cart and checkout flow
- Order lifecycle with status transitions
- Stock validation and rollback on cancellation
- REST API consumed by a JavaScript frontend
- Unit and integration-oriented testing with JUnit 5 and Mockito

## Tech Stack
- Java 17
- Spring Boot 3
- Spring Security
- Spring Data JPA
- PostgreSQL / H2
- MapStruct
- Maven
- JUnit 5 / Mockito
- JaCoCo
- Postman
- HTML / CSS / JavaScript
